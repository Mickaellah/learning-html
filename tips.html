Tips for writing a good code.

How to write HTML quicker.

  • Use a code editor that has auto-completion. Auto-completion allows you to start typing a tag or an attribute and the software predicts what you are typing and finish it for you.
  • Try to not use your mouse all the time, there are lots of keyboard shortcuts which help you to type faster and also to save your time.
