Attribute rules.

There are several attribute rules in html. If you want to write good, clean code you have to apply the attributes rules to make it looks good and easy to read.

Those are the attribute rules which I know :

  • We are not allowed to use capital letters in an attribute name, especially at W3School we always use lowercase attribute name.
  • In general, attribute value should have quotes to wrap around.
  • Normaly, attribute value itself contains double quote are not allowed, use single quote instead.
  • Attributes are separated with spaces.
  • Public attributes can go on an element.
  • You can only have dashes in the attribute name.
  • Attributes must have name value pairs.
