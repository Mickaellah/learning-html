Elements we can use in html5.

First let us see what an HTML is and what does it used for.

Hypertex Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as CSS (Cascading Style Sheets) and scripting languages such as Javascript. HTML elements are delineated by tags, written using angle brackets.

What are HTML elements and what do they do?

An HTML element is a type of HTML (Hypertext Msrkup Language) document component, one of several types of HTML nodes (there are also text nodes, comment nodes and others). Each element can have HTMl attributes specified. Elements can also have content, including other elements and text.

We are going to see some of the elements like:

  1. Inline elements.
  2. Void elements.
  3. Sectioning elements.
  4. Form elements.
  5. Head elements.
  6. Table elements.
  7. Block elements.

Inline elements.

Inline elements are those which only occupy the space bounded by the tags defining the element, instead of breaking the flow of the content.

  • <b> is for bold.
  • <a> is for anchor or link.
  • <i> is for italic.
  • <strong> is quite the same as <b>, but strong is used to stress a word or sentence.
  • <q> is for putting quote around a word or a sentence.
  • <em> is used to emphasize a word or a sentence.
  • <small> is for representing side-comments and small print.
  • <sub> is for adding subscript.
  • <sup> is for adding superscript.
  • <span> is for phrasing content.
  • <u> is for underline.
  • <ins> is used to identify text that has been inserted.
  • <del> represents text that has been deleted.
  • <code> represents a fragment of computer code.
  • <mark> used to highlight words.
  • <kbd> is used to show or represent keyboards shortcuts.
  • <var> represents the name of a variable in a mathematical expression.
  • <samp> is used to enclose inline text.
  • <s> is used to show text that is no longer accurate or relevant.
  • <dfn> represents the defining instance of a term.
  • <time> is used to represent a precise date.
  • <abbr> represent an abbreviation or acronym.
  • <address> represents the contact information.
  • <cite> is use to show a title of a work.
