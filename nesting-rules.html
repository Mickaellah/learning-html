Nesting rules

There are some rules about the nesting elements in html. HTMl element can contain other elements which wre called nested elements. If you want to make your code an eye catching code, you have to follow the rule about nesting elements. Nesting elements provide better functionaly and a neater look.

Rules need to follow in nesting elements.

  • Make sure you close the tags that you use in the right order.
  • The element which has opened last must be the first to close.
  • You have to close all of tags that can be closed before you open another one.
